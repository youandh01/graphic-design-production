# Accessibility Text

Use for web, SNS, public campaigns, reports, and presentations.

Classify the image purpose:

- Decorative
- Informative
- Functional
- Image of text
- Complex

Rules:

- Decorative image in HTML: empty alt text, `alt=""`.
- Informative image: describe the essential meaning.
- Functional image: describe the action or destination.
- Image of text: include the text unless repeated nearby.
- Complex image: provide short alt plus longer adjacent description.

Pattern:

```text
Alt text:
[Purpose] graphic showing [confirmed subject/action] in [confirmed context].
```
