# Adobe Delivery Checklists

## Ask First

- Is this for print, web/SNS, app, presentation, merchandise, or source handoff?
- What size, ratio, and orientation are required?
- Which source format is needed: PSD, AI, PDF, SVG, PNG, JPG?
- Is editability required, or only final output?
- Are brand fonts, logos, linked images, or printer specs provided?

## Photoshop

- Keep editable source layers when future revision is likely.
- Name key layers/groups clearly.
- Preserve a high-resolution master before exporting flattened files.
- Confirm color space: sRGB for most web/SNS, CMYK only when print workflow requires it.
- Embed or preserve ICC profile when color-managed delivery matters.
- Export layers/artboards when assets need separate files.
- Avoid repeated JPEG re-saving from already compressed JPEGs.

## Illustrator

- Check artboards and naming.
- Check linked images and missing links.
- Check fonts and whether text should remain editable or be outlined.
- Package files when handing off editable Illustrator work.
- Include links, package report, and font handling notes when allowed by license.
- For SVG, confirm text/outlines, styling, and embedded or linked images.

## Print

- final trim size
- bleed amount
- safe margin
- crop/trim marks if required
- CMYK or printer-requested color profile
- image resolution
- overprint/spot color requirements
- fonts outlined or embedded as required
- linked images included
- print PDF preset or PDF/X requirement if specified

Do not assume bleed or PDF/X without printer guidance.

## Web / SNS

- aspect ratio and pixel size
- file format
- file size limit
- transparent vs white background
- sRGB conversion
- readable text at mobile size
- platform safe area
- alt text or image description
